Pyjamas has its own logging module that you can use to display log messages with JavaScript's alert() function, printed in a dedicated div element in the HTML document, or written to the console. 

from pyjamas import logging
log = logging.getAlertLogger()   # other loggers: Append, Console, Print ...
...
log.error("Hello, here is an error")

To make your life even easier there's also a convenience module that provides an AppendLogger with simple text output already by default:

from pyjamas import log
log.info('Show this text and value %s now', value)

For a good understanding of the logging module read the Python Logging HOWTO; most of it directly applies to Pyjamas as the module is built on top of Python logging. Loggers you'll find useful:

  1. AlertLogger: log = logging.getAlertLogger()
    shows a browser alert popup dialog for each log message.
  2. AppendLogger: log = logging.getAppendLogger()
    appends text to the end of the HTML document body. The div element holding the log messages can be accessed by its element ID ('logging_logger_name', 'logging_pyjs' by default) for styling.
  3. ConsoleLogger: log = logging.getConsoleLogger()
    passes log messages on to the console of Firebug/Chrome/Opera using the console.log() function.
  4. PrintLogger: log = logging.getPrintLogger()
    prints text to cerr, the default error output stream. This is a good choice when developing/testing with Pyjamas Desktop.
  5. NullLogger: log = logging.getLoggerForHandler(NullHandler())
    disable logging completely and safely. Import this handler with from pyjamas.logging.handlers import NullHandler.

Note that for the ConsoleLogger (and the PrintLogger when used in a browser) you will need Firefox with Firebug installed, the Chrome browser, or Opera. If you haven't installed Firebug, and you write a log message (or use "print") you will get an error (because there will be no object called "console" in your browser's global namespace). So make your life easier, and install Firebug.
